## ⚡ 프로젝트 실행 방법 (How to Run)

단축키 Ctrl + `를 누르거나, 상단 메뉴에서 [터미널] -> [새 터미널]을 클릭하세요.

### 1. 파이썬 라이브러리 추가하기

```Bash
pip install -r requirements.txt
```

### 2. Streamlit 시작하기

```
streamlit run app.py
```
